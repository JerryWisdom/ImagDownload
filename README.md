# ImagDownload
C++ 网络图片下载器
=====

    代码使用C++语言实现。
    通过win下的SOCKET编程实现对服务器的请求访问，通过正则表达式解析出HTML中的图片链接，
    然后采用广度优先搜索（BFS）实现页面网址的遍历，效果还算不错。
    
    这份代码比较简短，依然存在可以优化的地方，如果有好的优化方法或者可以扩充功能，
